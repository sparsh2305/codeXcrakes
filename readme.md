# CodeXCrakes - Baby3 Lost & Found Application

## Overview
This repository contains the Baby3 Lost & Found application - a campus-wide lost and found portal.

## Features
- Lost & Found item reporting
- Admin dashboard for managing items
- Recycle bin for deleted items
- Dark/Light theme support
- Search and filtering capabilities

## Deployment
The Baby3 application is automatically deployed to GitHub Pages when changes are pushed to the main/master branch.

### Live Site
Once deployed, the application will be available at:
`https://sparsh2305.github.io/codeXcrakes/`

### Manual Deployment
To manually trigger a deployment:
1. Go to the Actions tab in GitHub
2. Select "Deploy Baby3 to GitHub Pages" workflow
3. Click "Run workflow"

## Local Development
1. Clone the repository
2. Navigate to the `baby3` directory
3. Open `index.html` in a web browser
4. Default login credentials:
   - Username: `admin`
   - Password: `admin123`

## Project Structure
```
baby3/
├── index.html           # Main application page
├── login.html          # Login page
├── admin-dashboard.html # Admin dashboard
└── recycle-bin.html    # Recycle bin view
```

## Technologies Used
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- Chart.js for analytics
- LocalStorage for data persistence
